# Exoplanet Pipeline (Hackathon Modular Version)
## Run
From this folder:

```bash
python exoplanet_pipeline.py
```

Or directly:

```bash
python -m exoplanet_pipeline_mod.main
```

## Outputs
The pipeline writes to `OUT` (see `config.py`):
- `pipeline_results.json`
- `exoplanet_pipeline_report.pdf`
- `summary_grid.png`
- `plots/*.png`
